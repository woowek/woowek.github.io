> 프로젝트 개요
---
- 경기도청에 있는 대상 서버들의 시스템을 P2V, V2V, V2C 의 형태로 서비를 전환한다.
- VM과 컨테이너의 경우 오케스트로의 viola로 구성하며 VM은 openstack, 컨테이너의 경우 kubernetes로 구성한다.
- 나는 이 중 V2C 즉 컨테이너 전환을 수행한다.


- 대상 구성은 다음과 같다.
    * apache wordpress
    * egovframework


- 작업 내용을 크게 구분지으면 다음과 같다.
    * application 분석
    * 저장소 구성
    * 컨테이너 dockerfile 구성
    * k8s 구성요소 (deployment, service, ingress, pv/pvc) 구성
