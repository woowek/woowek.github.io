## 📚 주요 섹션

### Documents
왼쪽 메뉴에서 **Documents**를 클릭하거나 [여기](/docs/documents/)를 클릭하면 `woowek/documents` repository의 모든 마크다운 문서를 볼 수 있습니다.

- 📂 자동으로 모든 마크다운 파일을 검색합니다
- 📄 클릭하면 렌더링된 내용을 볼 수 있습니다
- 🔄 실시간으로 repository에서 최신 내용을 가져옵니다

---

## 🚀 기능

- **실시간 동기화**: GitHub repository의 최신 내용을 실시간으로 표시
- **마크다운 렌더링**: 코드 블록, 테이블, 이미지 등 모든 마크다운 요소 지원
- **검색 가능**: 사이트 전체 검색 기능 지원
- **다크 모드**: 편안한 읽기를 위한 다크 테마
